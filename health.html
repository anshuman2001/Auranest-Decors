How to read this:
🟢 Healthy — responding correctly.   🟠 Reachable but degraded — endpoint is alive but returned a configuration/validation error (e.g. missing env var, invalid test payload — expected for safe probes).   🔴 Down — unreachable or server error.

API probes use harmless test payloads, so an OTP/payment endpoint returning 400/502/503 still counts as online — it proves the function is deployed and running. Only network failures or 404/5xx-crash mean a real outage.
